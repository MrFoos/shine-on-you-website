# Admin-guide — shineonyou.no

**Innlogging:** https://shineonyou.no/admin

**Arrangementer** — Legg til/rediger/slett konserter. Sett billettstatus og eventuell billettenke.

**Om bandet** — Rediger biografitekst og klikk Lagre.

**Bandmedlemmer** — Legg til med navn, rolle og bilde. Klikk på navn/rolle for å redigere. Dra for å sortere.

**Videoer** — Legg til via YouTube-ID (koden etter `v=` i URL-en). Dra for å sortere.

**Galleri** — Last opp ett eller flere bilder. Dra for å sortere.

**Press Kit** — Kommer snart.

**Innstillinger** — Endre overskrifter, fotokreditt og lenker til sosiale medier.

---
Alle endringer vises umiddelbart på nettstedet. Sletting krever alltid bekreftelse.
